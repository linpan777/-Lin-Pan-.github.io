## 如何啟用 GitHub Pages
1. 到本 repository 的 GitHub 頁面 → Settings → Pages。  
2. 在「Source」選擇 main branch（或 gh-pages），通常 root（/）或 /docs。  
3. 點擊 Save，等待幾分鐘並取得顯示的 Pages URL。  
4. 將 Pages URL 填回上面的「GitHub Pages URL」欄位。

## 如果使用 Sphinx（選用）
- 建議步驟（本地建置）：
  - pip install -U sphinx
  - sphinx-quickstart docs
  - 在專案根目錄執行：sphinx-build -b html docs docs/_build/html
- 將 docs/_build/html 的輸出內容放到 GitHub Pages 指定的 branch 或 /docs 資料夾。
